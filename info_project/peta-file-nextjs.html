💡
Tip: Tambah Proyek Baru

Untuk menambahkan proyek baru, kamu hanya perlu mengedit src/data/projects.jsx saja. Tidak perlu menyentuh file halaman sama sekali — data akan otomatis muncul di halaman /projects dan detail proyeknya.

⚠️
Jangan lupa tambahkan .next/ ke .gitignore

Folder .next/ adalah build cache yang digenerate otomatis oleh Next.js. Sebaiknya ditambahkan ke .gitignore agar tidak ikut ter-push ke GitHub setiap kali kamu build.
